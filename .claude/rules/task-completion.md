# タスク完了時のチェックリスト

タスクを完了する前に、以下を確認すること。

## 必須チェック

| 順序 | コマンド            | 確認内容                                           |
| ---- | ------------------- | -------------------------------------------------- |
| 1    | `npm run typecheck` | 型エラーなし                                       |
| 2    | `npm run lint`      | ESLintエラーなし（`npm run lint:fix`で自動修正可） |
| 3    | `npm run test`      | テストパス（該当する場合）                         |
| 4    | `npm run build`     | ビルド成功（重要な変更の場合）                     |

## 条件付きチェック

| 条件               | 必要な対応                                                   |
| ------------------ | ------------------------------------------------------------ |
| Prismaスキーマ変更 | `npx prisma generate` → `npm run typecheck`                  |
| 新規ページ追加     | `app/(main)/`配下に配置、ミドルウェア保護確認                |
| 環境変数追加       | `src/env.ts`にバリデーション追加                             |
| 新機能追加         | テストを`__tests__/`に作成（`.claude/rules/testing.md`参照） |

## 注意事項

- コミットは明示的に依頼された場合のみ実行
- セキュリティベストプラクティスを遵守
